# 일반적인 프로그래밍 원칙
## 57. 지역변수의 범위를 최소화하라.
### 핵심 요약
+ 지역변수의 범위를 줄이는 가장 강력한 기법은 역시 `가장 처음 쓰일 때 선언하기`다.  
	+ 사용하려면 멀었는데, 미리 선언부터 해두면 코드가 어수선해져 가독성이 떨어진다.  
+ 거의 모든 지역변수는 `선언과 동시에 초기`해야 한다.
	+ 초기화에 필요한 정보가 충분하지 않다면 충분해질 때까지 선언을 미뤄야 한다.  
+ 메서드를 작게 유지하고 한 가지 기능에 집중하자.
